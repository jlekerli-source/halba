### Description

Halba is a local-first trust operations control plane for AI-assisted work.

When agents say work is done across multiple local workspaces, Trust Inbox ranks the contradictions, stale proof, failed guards, changed evidence, and expired decisions that need attention now. Proof Mode then opens the exact claim, source, deterministic authority, and accountable human action.

Language proposes. Evidence decides. The human remains accountable.

Codex built the Build Week implementation delta, including Proof Mode, the eval corpus, the rendered interface, and the public release path. Halba's public proof packet includes a safe version of the Codex-authored change report and receipts that the product inspects.
